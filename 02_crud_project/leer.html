Ejemplo de como realizar un CRUD en Python y Django en Bootstrap

En el menú superior podremos ver las diferentes opciones que existen para la manipulación de datos mediante formularios tipos.

El ejemplo a desarrollar constará de:

  • Crear un usuario (Create)
  • Leer o ver el usuario creado (Read)
  • Actualizar o modificar datos del usuario creado (Update)
  • Eliminar el o los usuarios creados y todos sus datos (Delete)

El mini proyecto estará soportado en una Base de Datos MySQL
